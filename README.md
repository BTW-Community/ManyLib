# ManyLib for Better Than Wolves Community Edition (3.0.x)

---

## Features

- Aims to enable developers to easily read and write configuration settings to JSON files, and provides a simple in-game screen for adjusting those settings.

---

## Requirements

- Minecraft with [Better Than Wolves Community Edition 3.x](https://modrinth.com/mod/btwce)

---

## How to Install

1. [Download the latest release (.jar)](https://github.com/BTW-Community/ManyLib)
2. Place it into your `mods` folder
3. Start Minecraft with BTW Community Edition
